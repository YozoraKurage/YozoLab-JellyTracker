# NiMH 3.3V昇圧モジュール KiCADデータ
Ni-MHを3.3Vへ昇圧するモジュールです。

## How to Upload to JLCPCB
jlcpcb/production_filesに出力されたBOM、CPL、GERBERデータがあるので、それを使うかKiCADからデータを作成してください
